# MLA 9th Edition — Electronic & Web Source Formats

**Authority:** https://www.owl.purdue.edu/owl/research_and_citation/mla_style/mla_formatting_and_style_guide/mla_works_cited_electronic_sources.html  
MLA Handbook, 9th Edition (Modern Language Association, 2021)

---

## Core Format (Webpage with Author)

```
Last, First. "Title of Page." Title of Website, Publisher (if different from site), Day Mon. Year, URL.
```

Elements in order (MLA calls these "core elements"):
1. Author
2. Title of source (in quotation marks for pages within a larger work)
3. Title of container (italicized — the website, journal, or database)
4. Other contributors (editor, translator, etc.)
5. Version
6. Number (volume, issue)
7. Publisher
8. Publication date — Day Month Year (e.g., 15 May 2023)
9. Location (URL, DOI, or page numbers)

---

## All Cases

### Standard: Webpage with Individual Author
```
Poncelet, Barbara. "Mom Am I Fat?: Helping Your Teen Have a Positive Body Image." Verywell.com, About Inc., 20 Apr. 2016, www.verywell.com/mom-am-i-fat-3200843.
```

### Standard: Webpage, No Author — Title First
```
"Python (Snake)." Funk & Wagnalls New World Encyclopedia, Jan. 2018, p. 1. EBSCOhost, research.ebsco.com/linkprocessor/plink?id=20ec1dc2-2546-3225-9fdf-f89e28bcb6c5.
```

### Website with No Author, No Separate Page Title
```
"Title of Specific Page." Name of Website, Publisher, Date, URL.

"Climate Change Facts." NASA, NASA, 12 Mar. 2024, climate.nasa.gov/evidence/.
```

### With Access Date (use for content that changes or has no publication date)
```
Last, First. "Title." Website Name, Date, URL. Accessed Day Mon. Year.

Lundman, Susan. "How to Make Vegetarian Chili." eHow, www.ehow.com/how_10727_make-vegetarian-chili.html. Accessed 6 Feb. 2009.
```

### Online Newspaper/Magazine Article
```
Last, First. "Title of Article." Publication Name, Day Mon. Year, URL.

Bernstein, Fred A. "This Is a Newsworthy Title." The New York Times, 14 Apr. 2023, www.nytimes.com/article.html.
```

### Journal Article in a Database (Two Containers)
Container 1 = the journal; Container 2 = the database

```
Last, First, and First Last. "Title of Article." Name of Journal, vol. #, no. #, Month Year, pp. first–last. Database Name, DOI or URL.

Yang, Ruby Pi-Ju, and Kimberly A. Noels. "The Possible Selves of International Students and Their Cross-Cultural Adjustment in Canada." International Journal of Psychology, vol. 48, no. 3, June 2013, pp. 316–23. ERIC, https://doi.org/10.1080/00207594.2012.660461.
```

### eBook in a Library Database (Two Containers)
Container 1 = the eBook; Container 2 = the database

```
Author. Title of Book. Publisher, Year. Database Name, URL or DOI.
```

### Tweet (Twitter/X)
```
Last, First [@handle]. "Full text of tweet up to 280 characters." Twitter, Day Mon. Year, URL.

Nguyen, Viet Thanh [@viet_t_nguyen]. "I could put on my headphones at the Chinese restaurant or I could listen to @barrymanilow sing 'Mandy.' I choose Mandy." Twitter, 19 Feb. 2019, twitter.com/viet_t_nguyen/status/1097899776571764736.
```

### Instagram Post
```
Last, First [@handle]. "Caption or description." Instagram, Day Mon. Year, URL.

Chabon, Michael [@michael.chabon]. "#rip Milton Glaser. I grew up in his work." Instagram, 28 June 2020, www.instagram.com/p/CB-E9gngVwo.
```

### YouTube / Online Video
```
Last, First, director. "Title of Video." Website, Day Mon. Year, URL.

OR for a channel:

Channel Name. "Title of Video." YouTube, Day Mon. Year, URL.
```

### Government/Institutional Webpage
```
"Title of Page." Name of Government Body, Day Mon. Year, URL.
```

## Container Logic

The container concept is MLA 9's key structural innovation. A container is any larger whole that holds the source.

**Single container** (webpage within a website):
```
Author. "Article." *Website*, Date, URL.
```

**Two containers** (article in journal, found via database):
```
Author. "Article." *Journal*, vol., no., year, pp. *Database*, DOI/URL.
```

**Rule**: Each container block must be complete before starting the next. Each container's information ends with a period.

---

## DOI vs URL Rules

| Situation | Use |
|---|---|
| DOI available | Always prefer DOI over URL |
| DOI format | https://doi.org/10.XXXX/XXXXX |
| No DOI, stable permalink | Use permalink |
| No DOI, no permalink | Use full URL (https:// acceptable in MLA 9) |
| URL domain only (no path) | Include as-is |

MLA 9 allows but does not require the `https://` prefix. Either `www.` or `https://` is acceptable.

---

## Access Date Rules

| Situation | Include access date? |
|---|---|
| Stable, archived page | No |
| Page with no publication date | Yes |
| Wiki or frequently updated page | Yes |
| Standard news article | No |
| Social media post | Optional but recommended |

**Format:** `Accessed 15 May 2026.` — placed at end of entry, after URL.

## Punctuation Rules

- Periods end each core element block
- Commas separate elements within a container
- Colons introduce subtitles
- Quotation marks around article/page titles
- Italics for container/site/journal titles
- No period after a URL or DOI (end of entry)

---

## Validation Checklist (MLA9 Electronic)

For `valid` status, an electronic source entry must have:
- [ ] Author (or title-first if no author)
- [ ] Quoted title of the specific page or article
- [ ] Italicized title of the website or first container
- [ ] Publication date (or access date if no publication date)
- [ ] URL or DOI
- [ ] Second container block complete (if source found in a database)
